Coin Economy

Earn coins from activities. Spend in the Tree Shop.

    Log activities (sets, openings, closings, closed)2 coins per activity + bonus for daily target
  2. 2
    Hit 100% daily pledge = tree grows + 5 bonus coins
  3. 3
    Spend coins in Tree Shop for new species100-2,000 coins per tree
