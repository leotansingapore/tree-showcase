How It Works

Six proven engagement loops borrowed from the best habit-gamification apps

Coin Economy

Earn coins from activities. Spend in the Tree Shop. Each species has a price tier.

  1. 1
    Log activities (sets, openings, closings, closed)Earn 2 coins per activity + bonus for daily target
  2. 2
    Hit 100% daily pledge = tree grows + 5 bonus coinsCore loop: hit target, grow tree
  3. 3
    Spend coins in Tree Shop to unlock new speciesPrice range: 100-2,000 coins
Ref: Forest App - 4 base coins + bonus per focus session

Biome Variants

Each species x 5 color variants = 120+ collectibles without 5x art. Inspired by Habitica's species x potions grid.

Standard Frost Golden Sakura Phantom

Standard = shop. Frost = winter streak. Golden = top 10%. Sakura = spring event. Phantom = Halloween.

Mystery Seeds

Weekly pledge target = random Mystery Seed with rarity weights. Variable reward = dopamine.

Wilting (Loss Aversion)

Miss daily target? Your newest tree wilts (turns grey). Revive it by hitting next day's target. Softer than Forest (where trees die permanently).

Ref: Forest, Flora, SleepTown all use loss/destruction

Real Trees, Real Impact

Every 50 virtual trees planted = 1 real tree via Trees for the Future. Forest App planted 2M+ real trees this way. Users feel their daily grind has genuine environmental impact.
